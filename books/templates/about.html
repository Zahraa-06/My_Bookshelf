Personal Book Tracker is built using industry-standard technologies to ensure reliability, security, and performance:

Django Framework Robust Python web framework for the backend
SQLite Database Reliable data storage for your book collection
Responsive CSS Modern styling that works on all devices
User Authentication Secure login system to protect your data

About the Developer

This application was developed as part of a software engineering project to demonstrate full-stack web development skills using Django. The focus was on creating a clean, user-friendly interface while implementing solid backend architecture and database design.

The project showcases CRUD operations, user authentication, responsive design, and modern web development best practices.
